# MonuFind
Computer vision project to recognize monuments with the bag of world

# Dataset
Photos of the monuments of the city of Cagliari (Sardinia, Italy)
- Bonaria
- Bastione
- Basilica
- Caserma carabinieri
- Carlo Felice
- Palazzo delle Scienze
- Porta Cristina
- Torre dell'Elefante
- Municipio

# Tool/Coding
- OpenCV
- Python
